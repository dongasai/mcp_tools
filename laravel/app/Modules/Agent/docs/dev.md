## 当前开发重点：Agent提问功能

### 功能概述
Agent提问功能允许AI Agent在执行任务过程中主动向人类用户提出问题，获取指导、确认或澄清。

### 设计要点
- **问题类型**：选择类(CHOICE)、反馈类(FEEDBACK)
- **优先级**：紧急、高、中、低
- **状态管理**：待回答、已回答、已忽略
- **MCP集成**：通过Tool Actions实现
